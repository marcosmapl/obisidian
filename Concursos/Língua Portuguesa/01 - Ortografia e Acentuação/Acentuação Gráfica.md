## Acentuação segundo Sílaba Tônica

**Sílaba tônica:** sílaba pronunciada com maior intensidade.

- **Classificação quanto à tonicidade:**
	- **[[Monossílabos Tônicos]]**: e uma só sílaba pronunciadas com força (intensidade), tendo significado próprio e autonomia fonética.
    - **[[Oxítona]]:** palavra cuja **sílaba tônica (mais forte) é a última**
    - **[[Paroxítona]]:** palavra cuja **sílaba tônica (mais forte) é a penúltima**
    - **Proparoxítona:** palavra cuja **sílaba tônica (mais forte) é a antepenúltima**. Todas são acentuadas.

## Quadro-resumo

| Classificação        | Quando acentua                        |
| -------------------- | ------------------------------------- |
| Monossílabos tônicos | a(s), e(s), o(s)                      |
| Oxítonas             | a(s), e(s), o(s), em(ens)             |
| Paroxítonas          | terminações especiais (≠ a, e, o, em) |
| Proparoxítonas       | **sempre**                            |

## Acentuação do Hiato

Para fins de acentuação, importa distinguir:
- **Vogal:** som forte (tônico)
- **Semivogal:** som fraco (átono)

💡**Exemplo comparativo:**
- _ca-í_ → **í** é vogal tônica → hiato
- _cai_ → **i** é semivogal → ditongo

✍️ A acentuação do **[[Hiato]]** tem a função de **impedir a leitura como ditongo**.
