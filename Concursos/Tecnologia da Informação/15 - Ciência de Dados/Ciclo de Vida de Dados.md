## Ciclo Básico

![[Pasted image 20260527101646.png]]

### Etapa de Coleta

- Qual é o **escopo da necessidade da informação**?
- Que tipo de **resultado se espera**?
- Com quais **características**? 
- Quais são os **dados necessário**s? 
- Onde estão as **fontes** para estes dados? 
- **Como os dados podem ser coletados**? 
- Em que **formato** estão?

### Etapa de Tratamento

- A etapa de tratamento buscará tratar, transformar, processar, limpar, padronizar e sanitizar os dados obtidos de diferentes fontes de acordo com o padrão de qualidade predefinido.

### Etapa de Armazenamento

- Essa etapa busca processar, transformar, inserir, modificar, carregar, migrar, transmitir ou qualquer ação semelhante que vise persistir os dados em um dispositivo.
-  Quais são os dados disponíveis? 
- Quais destes dados serão armazenados? 
- Qual estrutura (física e lógica) será utilizada para seu armazenamento?

### Etapa de Integração

- A etapa de integração busca integrar dados de diferentes fontes em uma base de dados unificada.

### Etapa de Recuperação

- Essa fase busca encontrar, acessar, consultar e interpretar dados armazenados pelos usuários.

### Etapa de Descarte

- Essa etapa busca basicamente descartar os dados armazenados. 
- Quais dados já não são mais necessários? 
- Os dados a serem descartados foram persistidos? 
- Em quais suportes? 
- Estes dados estão replicados em outras bases?
