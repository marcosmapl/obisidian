# Dado

Dados são a **menor partícula estruturada** que compõe uma informação. São sucessões de fatos brutos, que não foram organizados, processados, relacionados, contextualizados, avaliados ou interpretados, representando apenas partes isoladas de eventos, situações ou ocorrências. 

Eles se constituem de unidades básicas a partir das quais informações poderão ser elaboradas ou obtidas. Além disso, eles são frequentemente quantificados, facilmente estruturados, facilmente transferíveis e facilmente obtidos por máquinas.

Quando os dados são organizados, processados, relacionados, contextualizados, avaliados ou interpretados, ele passa a ser uma **informação**, conforme veremos no tópico seguinte.

# Informação

A **informação** é a ordenação e organização dos dados de forma que passa a transmitir uma mensagem compreensiva dentro de um determinado **contexto**.

Entre os métodos utilizados para transformar dados em informação, temos:
- **Contextualização**
- **Categorização**
- **Cálculos**
- **Correção**
- **Sumarização**

Existe tanto um **ciclo de vida dos dados** quanto um **ciclo de vida da informação**.

# Conhecimento

Basicamente, o **conhecimento** acontece quando a **informação é aplicada**. Conhecer é o processo de compreender e interiorizar as informações recebidas, possivelmente combinando-as de forma a gerar mais conhecimento.

# Inteligência

A **inteligência** (ou sabedoria) é quando você utiliza o seu **conhecimento** para algum **propósito**. A inteligência tem uma premissa interessante: deve ser utilizada com algum propósito, isto é, para resolver algum problema. A inteligência envolve a capacidade de julgamento (Ex: Certo/Errado, Bom/Ruim, etc), isto é, usar o conhecimento para tomar a melhor decisão ou aplicá-lo a alguma situação concreta.
