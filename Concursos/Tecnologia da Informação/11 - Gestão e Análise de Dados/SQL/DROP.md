## Conceito

> [!note] DROP
> Remove **definitivamente** um objeto do banco de dados.

#tecnologia_informacao/sql_ansi/ddl/drop

> [!warning] A estrutura e os dados são perdidos.

---
## Exemplos

### Remover tabela

```
DROP TABLE aluno;
```

### Remover visão

```
DROP VIEW vw_alunos;
```
