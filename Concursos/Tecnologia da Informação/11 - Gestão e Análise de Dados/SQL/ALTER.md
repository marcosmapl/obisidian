## Conceito

> [!note] ALTER
> Modifica a estrutura de um objeto existente.

#tecnologia_informacao/sql_ansi/ddl/alter

---
## Operações comuns

- Adicionar coluna
- Alterar tipo de dado
- Remover coluna

---
## Exemplos

### Adicionar coluna

```
ALTER TABLE aluno ADD email VARCHAR(150);
```

### Remover coluna

```
ALTER TABLE aluno DROP email;
```
