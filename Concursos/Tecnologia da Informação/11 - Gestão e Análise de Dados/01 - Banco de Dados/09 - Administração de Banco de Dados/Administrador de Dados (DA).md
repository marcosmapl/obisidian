**Administrador de Dados (DA):** Tem uma visão mais _estratégica e de negócio_ sobre os dados. É responsável por definir políticas, padrões, qualidade, significado (semântica) e arquitetura dos dados corporativos. Foca no "o quê" e "porquê" dos dados.
    - Planejamento de dados corporativos.
    - Modelagem de dados conceitual e lógica.
    - Definição de padrões e nomenclaturas de dados.
    - Gerenciamento de metadados e dicionário de dados.
    - Garantia da qualidade e segurança dos dados em nível corporativo.
    - Definição de políticas de acesso e uso dos dados.


> [!important]  **Atribuições específicas do DA:** modelagem de dados, padrões, qualidade dos dados, metadados, planejamento estratégico dos dados.
