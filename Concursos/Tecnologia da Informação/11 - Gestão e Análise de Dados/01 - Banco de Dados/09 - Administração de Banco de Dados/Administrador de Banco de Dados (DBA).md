**Administrador de Banco de Dados (DBA):** Tem um foco mais _técnico e operacional_, centrado no SGBD e na infraestrutura do banco de dados. É responsável pela implementação, manutenção, performance e segurança do ambiente de banco de dados. Foca no "como" os dados são gerenciados pelo SGBD.
    - Instalação, configuração e atualização do SGBD.
    - Projeto e implementação do banco de dados físico.
    - Monitoramento e otimização de desempenho (tuning).
    - Gerenciamento de armazenamento e capacidade.
    - Políticas de backup e recuperação de desastres.
    - Gerenciamento de segurança do SGBD (usuários, permissões, roles).
    - Resolução de problemas técnicos.
    - Garantir disponibilidade e integridade dos dados no SGBD.

> [!important]  **Atribuições específicas do DBA:** performance, backup, segurança do SGBD, instalação, esquema físico.
