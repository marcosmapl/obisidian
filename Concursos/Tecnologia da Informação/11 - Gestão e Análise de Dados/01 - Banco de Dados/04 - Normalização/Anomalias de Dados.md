## Conceito

> [!note ] Anomalia de dados
> É um **problema decorrente de uma modelagem inadequada** do banco de dados, no qual a **estrutura das tabelas gera comportamentos indesejados** durante as operações de **inserção, atualização ou exclusão**, comprometendo a **integridade, consistência e confiabilidade** das informações.

## Definição Sintética (Prova)

> [!tip] Anomalia de dados é a ocorrência de efeitos indesejados nas operações de manipulação de dados, provocados por redundâncias e dependências funcionais inadequadas na estrutura do banco de dados.
