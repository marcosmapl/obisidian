## Definição

> [!note] Independência Lógica de Dados
> Permite alterar o **esquema conceitual** sem afetar os **esquemas externos ou aplicativos**. 
> Por exemplo, é possível adicionar ou remover tipos de registros na base de dados sem impactar os esquemas externos que se referem apenas aos dados existentes.

> [!warning] Dependendo o tipo de alteração, ela poderá ser refletida nos níveis superiores.
