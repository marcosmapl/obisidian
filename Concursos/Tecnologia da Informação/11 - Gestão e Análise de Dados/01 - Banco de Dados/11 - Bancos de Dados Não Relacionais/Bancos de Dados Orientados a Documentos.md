## Conceito

Armazenam dados na forma de **documentos**, normalmente em formatos **JSON, BSON ou XML**.

Cada documento contém **dados e estrutura**, podendo ter campos diferentes entre documentos da mesma coleção.

**Características:**

- Estrutura flexível
- Ideal para dados semiestruturados
- Organização em **coleções de documentos**
- Consulta por atributos internos do documento

**Exemplos:**

- MongoDB
- CouchDB

**Uso comum:**

- Aplicações web
- APIs
- Sistemas de conteúdo
