## Conceito

Armazenam dados em **famílias de colunas** em vez de linhas completas.
Inspirados no **Google Bigtable**.

**Características:**

- Estrutura orientada a colunas
- Alta compressão de dados
- Excelente para **grandes volumes e consultas analíticas**
- Escalabilidade horizontal

**Estrutura conceitual:**

> [!important] Linha → Famílias de Colunas → Colunas

**Exemplos:**

- Apache Cassandra    
- HBase

**Uso comum:**

- Big Data
- Sistemas analíticos
- Data warehouses distribuídos
