## Conceito

> [!note] RENAME
> Renomeia objetos do banco de dados.

> [!warning] A sintaxe exata pode variar, mas o conceito existe no SQL padrão.

---
## Exemplo conceitual

```
ALTER TABLE aluno RENAME TO estudante;
```
