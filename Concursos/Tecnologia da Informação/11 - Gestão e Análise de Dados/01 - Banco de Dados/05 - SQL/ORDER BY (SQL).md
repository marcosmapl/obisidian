## Conceito

> [!note] Order By
> Por meio dela é possível ordenar os resultados em ordem **crescente** (**ASC**) ou **decrescente** (**DESC**).

> [!tip] O valor padrão da ordenação é **crescente** (**ASC**).

---
## Exemplos

### Ordem Crescente

```sql
SELECT NOME
FROM ALUNOS 
ORDER BY NOME ASC;
```
### Ordem Decrescente

```sql
SELECT NOME
FROM ALUNOS 
ORDER BY NOME DESC;
```
