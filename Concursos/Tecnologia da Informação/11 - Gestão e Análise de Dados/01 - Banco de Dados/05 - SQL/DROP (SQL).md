## Conceito

> [!note] DROP
> Remove **definitivamente** um objeto do banco de dados.

#tecnologia_informacao/sql/ddl/drop

> [!warning] A estrutura e os dados são perdidos.

---
## Exemplos

### Remover tabela

```sql
DROP TABLE aluno;
```

### Remover visão

```sql
DROP VIEW vw_alunos;
```
