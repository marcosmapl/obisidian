## Conceito

> [!note] Where
> Utilizado para fazer a **Seleção** (filtragem) de quais registros (linhas) por meio de **Condições**.

#tecnologia_informacao/sql/where

## Exemplos

### WHERE - SELECT

```sql
SELECT * FROM ALUNO WHERE CIDADE = 'Manaus';
```
