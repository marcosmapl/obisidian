Armazenam dados em formato binário.
- **BIT(n)**  
    Sequência binária de tamanho fixo.
- **BIT VARYING(n)**  
    Sequência binária de tamanho variável.
- **BLOB** (_Binary Large Object_)  
    Grandes volumes de dados binários (imagens, arquivos, etc.).
