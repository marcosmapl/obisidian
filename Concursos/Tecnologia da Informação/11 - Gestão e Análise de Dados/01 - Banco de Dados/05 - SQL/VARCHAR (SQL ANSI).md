## Conceito

> [!note] VARCHAR (SQL ANSI)
> O **VARCHAR** é um tipo de dado do padrão **SQL ANSI** utilizado para armazenar **cadeias de caracteres de comprimento variável**, até um limite máximo definido.

#tecnologia_informacao/sql_ansi/varchar

---
## Características

- Possui **tamanho máximo** definido na criação (VARCHAR(n));
- Armazena apenas os caracteres efetivamente informados, **sem preenchimento com espaços**;
- Mais **eficiente em armazenamento** para textos de tamanho variável;
- Amplamente utilizado para nomes, descrições e textos curtos.

---
## Exemplos

- `VARCHAR(50)` para nomes de pessoas;
- `VARCHAR(100)` para endereços de e-mail;
- `VARCHAR(255)` para descrições curtas.
