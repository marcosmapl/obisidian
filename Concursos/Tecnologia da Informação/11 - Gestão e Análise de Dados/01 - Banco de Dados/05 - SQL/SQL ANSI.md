## Conceito

> [!note] SQL
> É a linguagem padrão para manipulação de bancos de dados relacionais.

> [!important] Linguagem Declarativa
> SQL é uma linguagem **declarativa**, ela não especifica como os dados devem ser consultados no banco de dados, apenas especifica quais informações devem ser retornadas.

> [!important] Case Insensitive
> As palavras-chave que compõem a linguagem **SQL** são **Case Insensitive**, enquanto que os valores textuais são **Case Sensitive**, devendo ser colocados entre **aspas simples**.

## Sublinguagens SQL

SQL é uma linguagem que permite realizar determinadas operações em um banco de dados relacional por meio de um conjunto de comandos. Em geral, esses comandos são agrupados basicamente em quatro **sublinguagens:**
- [[DDL — Data Definition Language]] (Data Definition Language)
- [[DML — Data Manipulation Language]] (Data Manipulation Language)
- DTL
- DCL
