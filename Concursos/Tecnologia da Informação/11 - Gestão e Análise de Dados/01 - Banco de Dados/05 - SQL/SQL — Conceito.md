## Definição

> [!note] SQL (Structured Query Language)
> É a **linguagem padrão** para **definição, manipulação, controle e consulta de dados** em **Sistemas Gerenciadores de Bancos de Dados Relacionais (SGBDs)**, conforme o **padrão ANSI/ISO**.

#tecnologia_informacao/banco_dados/sql

---
## Finalidade

- Criar e estruturar objetos do banco de dados;
- Inserir, alterar, excluir e consultar dados;
- Controlar acesso e permissões;
- Garantir integridade e consistência das informações.

## Sublinguagens SQL

SQL é uma linguagem que permite realizar determinadas operações em um banco de dados relacional por meio de um conjunto de comandos. Em geral, esses comandos são agrupados basicamente em quatro **sublinguagens:**
- [[DDL — Data Definition Language]] (Data Definition Language)
- [[DML — Data Manipulation Language]] (Data Manipulation Language)
- DTL
- DCL

## Ponto de Prova

>[!tip] Linguagem **declarativa**, na qual o usuário informa **o que deseja** e não **como o SGBD deve executar**.
