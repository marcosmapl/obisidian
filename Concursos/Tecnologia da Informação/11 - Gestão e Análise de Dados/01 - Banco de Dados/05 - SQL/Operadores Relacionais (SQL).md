## Operadores

| Operador | Descrição      | Exemplo                      |
| -------- | -------------- | ---------------------------- |
| =        | IGUAL          | WHERE nome = 'marcos';       |
| >        | MAIOR          | WHERE salario > 100.0;       |
| >=       | MAIOR OU IGUAL | WHERE idade >= 18;           |
| <        | MENOR          | WHERE data < '2000-01-01';   |
| <=       | MENOR OU IGUAL | WHERE velocidade <= 100;     |
| <>       | DIFERENTE      | WHERE cidade <> 'são paulo'; |
