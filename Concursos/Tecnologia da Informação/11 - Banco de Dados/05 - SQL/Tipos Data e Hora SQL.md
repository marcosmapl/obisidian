## Tipos Data

Representam datas do calendário.

- **DATE**  
    Possui **10 posições**, no formato **DD-MM-YYYY**  
    Exemplo: `30-03-2019`

### Funções associadas:
- **DAY()**
- **MONTH()**
- **YEAR()**

---
## Tipos de Hora

Representam horários.

- **TIME**  
    Possui **8 posições**, no formato **HH:MM:SS**  
    Exemplo: `23:54:55`

### Funções associadas:
- **HOUR()**
- **MINUTE()**
- **SECOND()**
