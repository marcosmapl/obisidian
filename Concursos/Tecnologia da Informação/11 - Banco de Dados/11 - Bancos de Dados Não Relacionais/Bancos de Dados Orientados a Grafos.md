## Conceito

Modelam dados utilizando **grafos**, compostos por:

- **Nós (nodes)** → entidades
- **Arestas (edges)** → relacionamentos
- **Propriedades** → atributos

**Características:**

- Foco em **relacionamentos complexos**
- Consultas eficientes de conexões entre dados
- Estrutura altamente conectada

**Exemplo de representação:**

> [!important] 
> Pessoa → (AMIGO_DE) → Pessoa  
> Pessoa → (TRABALHA_EM) → Empresa

**Exemplos:**

- Neo4j    
- Amazon Neptune

**Uso comum:**

- Redes sociais
- Sistemas de recomendação
- Análise de fraude
