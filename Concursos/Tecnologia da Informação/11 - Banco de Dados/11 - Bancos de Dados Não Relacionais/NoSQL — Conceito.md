## NoSQL — Conceito

**NoSQL (Not Only SQL)** refere-se a uma categoria de bancos de dados **não relacionais**, projetados para lidar com **grandes volumes de dados**, **alta escalabilidade** e **flexibilidade de estrutura**.

**Características principais:**

- Não utilizam obrigatoriamente **tabelas e esquemas rígidos** como nos bancos relacionais.
- Permitem **dados sem esquema fixo (schema-less)**.
- São projetados para **alta escalabilidade horizontal** (scale-out).
- Suportam **grandes volumes de dados distribuídos**.
- Geralmente priorizam **performance e disponibilidade**.

**Motivações para uso:**

- Big Data
- Aplicações web de grande escala
- Sistemas distribuídos
- Dados não estruturados ou semiestruturados
