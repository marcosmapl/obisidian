## Conceito

> [!note] Distribuição de Frequências
> Série estatística na qual **permanecem constantes o fato, o local e a época**. 

---
## Classificações

Ela pode ser classificada em dois tipos:
- **Distribuição de frequências pontual (ou discreta)**
- **Distribuição de frequências intervalar (ou contínua)**

### Distribuição de frequências pontual

Na distribuição de frequências pontual, **são apresentados todos os dados coletados juntamente com suas respectivas frequências**, não havendo perda de valores.

| Tempo (Min) | Freq. |
| ----------- | ----- |
| 85          | 1     |
| 89          | 1     |
| 96          | 1     |
| 98          | 2     |
| 99          | 3     |
| 103         | 1     |
### Distribuição de frequências intervalar

Na distribuição de frequências intervalar, **são apresentados os dados agrupados por intervalos e classe e frequências correspondetes**, não havendo perda de valores.

| Intervalo      | Frequência (Fi) |
| -------------- | --------------- |
| 85 <= x < 100  | 5               |
| 100 <= x < 115 | 12              |
| 115 <= x < 130 | 10              |
| 130 <= x < 145 | 7               |
